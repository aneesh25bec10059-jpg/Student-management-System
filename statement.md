Problem Statement
I have noticed that in most schools and colleges the marks of students are written somewhere and then the teacher has to manually calculate everything. I thought this is very time consuming because every time totals and averages needs to be done again and again. Sometimes mistakes also happen and that can change the result. I have seen teachers struggling with this specially when there are many students in a class  . So I wanted to make something simple that can calculate marks automatically and also show how the   class is doing in one place.
Scope of the Project
In thiss project, I have planned to build a small grade management system which can help in calculating grades easily. I have keept the scope simple because it is mainly for basic use. The system can add students, take their marks and then calculate the aveerage and grade. I wanted to display the list properly and also show a graph so its easier to compare students. I have only focused on the core features and not advanced features because it’s not needed for this project level.
Target Users

•	Students who want to track their results
•	Tuition teachers .
