Grade Management System
Overview
This is a small desktop app made for managing studentss marks and grades.
It lets the user  add students , enter their marks and it will calculate the total, average and grade.
The system also shows a graph of the class performance based on average marks\
Features
•	Add student name and his marks
•	Automatic  total, average and grade
•	View saved student list.
•	Class  average comparison graph
•	Simple and clean UI
•	Enter key moves to next field.
Technologies Used
•	Python
•	Tkinter for  GUI
•	Matplotlib for  graph display.
How to Install & Run
1.	Install Python if not installed already.
2.	Install matplotlib.
3.	Download the project folder.
4.	Open the .py file.
5.	Run using: python main.py
How to Test / Use
1.	Start the application.
2.	Type student name.
3.	Type marks separated by space.
4.	Press Add student.
5.	Press “Show Class Performance Graph” to view chart.
6.	Compare averages of all students
